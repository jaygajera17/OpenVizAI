# openVizAI

Monorepo for the openVizAI project.

Structure
- `apps/` — applications (frontend will go here)
- `packages/` — shared packages (types, config, UI, etc.)

Getting started
1. Install dependencies at repository root:

```bash
npm run bootstrap
```

2. Start the backend (already configured):

```bash
npm run dev:server
```

3. Create a frontend in `apps/web` (Vite or CRA recommended)..
